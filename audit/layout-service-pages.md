# Layout des pages services WordPress — Structure exacte

## Couleurs du thème WordPress (DIFFÉRENTES du React actuel)
- Texte principal : #382d2d (marron foncé)
- Boutons CTA : #D96B2F (orange)
- Liens : #D35355 (rose/rouge)
- Fonds : blanc cassé / #ede2d7 (beige clair)
- Fonts : Inter (sans-serif) + Cardo (serif accent)

## Structure page service (du haut vers le bas)

### 1. HERO — Image plein écran avec texte superposé
- Image de fond full-width
- Texte centré par-dessus : titre + sous-titre + description
- Badge info : "1h | 60€ | Visio ou présentiel" (quand applicable)
- Bouton "Prendre RDV" (orange arrondi)

### 2. MES ACCOMPAGNEMENTS — Grille horizontale
- 6 cartes/liens en grille (3x2 ou 6x1)
- Service actuel mis en évidence
- Liens vers tous les services

### 3. TU HÉSITES ? — CTA pleine largeur
- Fond gris clair / beige
- Texte + bouton "Me contacter"

### 4. CONTENU DÉTAILLÉ (Phytothérapie uniquement)
- "La puissance douce du végétal" — 2 colonnes possible
- "Séance type & accompagnement"
- Listes à puces
- Titres en orange

### 5. FAQ — Accordéons
- Questions cliquables avec expand/collapse

### 6. RÉSERVER — Formulaire/CTA
- Section réservation pleine largeur

### 7. PROCHAINS ATELIERS — Cartes événements
- Grille de cartes avec images

### 8. TÉMOIGNAGES — Carousel
- Slider avec photos, citations, notes

### 9. MES PRATIQUES
- Texte + liens vers services

### 10. CITATION
- "Je fais ma part, j'ouvre mon cœur..."

### NOTE IMPORTANTE
- PAS de sidebar sticky — tout est full-width par sections
- La section "Mes Accompagnements" n'est PAS une sidebar mais une grille horizontale
